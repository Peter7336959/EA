職業噪音：不可逆的無聲威脅

深入了解工作場所的噪音危害，並學習如何透過系統化的聽力保護計畫保護勞工的聽覺健康。

一、噪音問題的嚴重性

📈

通報率第一

根據勞動部統計，噪音性聽力損失長年高居我國職業病通報量首位，佔總通報量超過40%，是職場最普遍的健康危害。

🚫

永久性傷害

噪音造成的內耳毛細胞損傷是不可逆的。一旦受損死亡便無法再生，這意味著「預防」是唯一的防線。

🌍

普遍性危害

從製造業、營造業到服務業，超過50種職業都潛藏噪音風險，影響著廣大的勞動人口。

噪音如何損傷聽力？

長期暴露在高分貝環境下，會對內耳中脆弱的毛細胞造成壓力，導致其損傷。聽力圖上最典型的特徵是在4000Hz附近出現一個明顯的凹陷(Notch)，這也是職業性聽損的關鍵指標。

二、聽力保護計畫 (HCP) 六大核心要素

一個有效的聽力保護計畫 (HCP) 是一個系統性的管理循環，旨在辨識、評估並控制工作場所的噪音危害，保護勞工免於聽力損失。以下為其六大核心要素，環環相扣，缺一不可。

📋

1. 噪音暴露評估

科學化地量測並評估勞工的實際噪音暴露量，以確認誰需要被納入計畫管理。

🛠️

2. 工程與行政管制

從源頭改善噪音問題，例如使用低噪音設備或設置隔音屏障，並調整工作流程以減少暴露。

🎧

3. 聽力防護具

當噪音無法有效降低時，提供並確保勞工正確佩戴合適的耳塞或耳罩。

🩺

4. 聽力檢查

定期進行聽力測試，建立基準並追蹤變化，以監測計畫成效並及早發現聽損警訊。

🎓

5. 教育訓練

教導勞工噪音危害、防護具使用及計畫內容，提升其自我保護意識與能力。

📊

6. 紀錄保存與評估

完整記錄所有執行過程，並每年定期評估計畫成效，作為持續改善的依據。

三、噪音控制層級金字塔

解決噪音問題應遵循「控制層級」原則，優先從最有效的方法著手，而非僅僅依賴個人防護。

工程控制
(源頭改善)
行政管理控制
(改變工作方式)
個人防護具
(最後防線)

四、台灣法規關鍵數值

《職業安全衛生設施規則》明確定義了雇主應採取的行動標準，了解這些數值是執行計畫的基礎。

五、成效監測 - 聽力閾值變化

聽力保護計畫的成功與否，可透過追蹤勞工聽力檢查結果來評估。若年度聽力圖與基準相比出現「顯著聽力閾值變異 (STS)」，即代表保護措施可能不足，需立即檢討並介入。
