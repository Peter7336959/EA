CH13

第13章：混合物暴露評估與IHMOD 2.0 (單頁摘要)

應用IHMOD 2.0軟體，評估在車庫內傾倒含2-丁氧基乙醇(2BE)的擋風玻璃清洗液所產生的消費者暴露風險。此評估的核心挑戰在於該清洗液為非理想的化學混合物，而非純物質。

關鍵挑戰：計算混合物生成率 (G)

對於水與2BE的非理想混合物，不能直接使用傳統的拉午耳定律，否則將嚴重低估風險。本案例的關鍵是引入活性係數 (γ) 來校正分子間的交互作用力。透過UNIFAC方法計算，2BE在此水溶液中的活性係數高達 7.31，這表示其真實蒸發趨勢是理想模型預測的7倍以上。經此校正後，計算出的蒸氣生成率 (G) 約為 20 mg/min，此數值是後續暴露模擬的關鍵輸入。

模型與分析方法

為模擬污染物在空間中的濃度梯度，本評估採用了比傳統「完全混合模型」更真實的雙區（近場／遠場）模型，將空間劃分為作業人員呼吸區周圍的「近場」與其餘空間的「遠場」。

運用IHMOD 2.0軟體進行了兩種分析：

  1. 決定論分析：使用單一「點估計值」輸入參數，得到一個特定的預測結果。
  2. 機率性（蒙地卡羅）分析：為關鍵輸入參數（如生成率、通風率）設定機率分佈，進行上萬次模擬，從而得到一個暴露濃度的可能性分佈，更完整地體現真實世界的不確定性。

核心結果與風險判斷

兩種分析方法都聚焦於作業期間（前30秒）的近場峰值濃度：

  • 決定論峰值濃度：預測為 1.98 mg/m³。
  • 機率性第95百分位數峰值濃度：作為「合理最壞情況」的估計，預測為 2.5 mg/m³。

在風險判斷上，將這些預測值與健康基準比較：

  • 職業暴露限值 (OELs)：兩種方法的預測結果均遠低於最嚴格的NIOSH建議限值 (24 mg/m³)。
  • 環境參考濃度 (EPA RfC)：預測結果超過了針對一般民眾終身暴露所設的RfC值 (1.6 mg/m³)。

結論

綜合判斷，對於單次、短暫的消費者使用情境，此產品的暴露風險低且可接受。然而，由於預測濃度超過了慢性的健康基準(RfC)，對於需要每日頻繁操作的職業人員（如汽車維修技師），其長期風險則需進一步的審慎評估。

本案例完整展示了IHMOD 2.0作為現代工業衛生工具的價值：它不僅能處理複雜的混合物情境，實現前瞻性的「設計安全」評估，更能透過蒙地卡羅模擬來量化不確定性，為風險管理提供更穩健、更具信服力的科學依據。
