週期性清潔作業暴露風險評估

案例研究：使用醋酸溶液之清潔作業

評估情境：日常作業的潛在風險

一名作業員每三分鐘使用含4%醋酸的清潔液擦拭一塊金屬面板。我們需要評估：他們呼吸的空氣是否安全？本研究透過三種數學模型，深入分析其暴露風險。

暴露模型分析：三種視角，一個結論

模型一：WMB穩定快速蒸發

假設醋酸瞬間完全蒸發。這是一種簡單且保守的估算方法，常用於初步風險篩選。

模型二：WMB重複性累加暴露

模擬蒸發率隨時間遞減的真實情況，顯示濃度在工作班次中如何逐漸累積。

模型三：WMB穩態平均

一種估算長期平均濃度的簡便方法，無需複雜的時變模擬，即可得到可靠結果。

最終風險評估

本案例透過三種不同複雜度的理想混合箱(Well-Mixed Box, WMB)模型進行評估。結果一致表明，即使在最保守的假設下，作業員的醋酸暴露量也顯著低於職業暴露限值。模型二因考慮了更真實的時變與累積效應，其結果(19% OEL)被視為最接近實際情況的估算，為此作業的安全性提供了強力的數據支持。

結論：此清潔作業安全。所有模型均確認，作業員的醋酸暴露濃度雖然遠低於法規限值，但仍顯著高於公司內部設定的 10% 行動基準（2.5 mg/m3）。

所以，EHS 部門決定「安排此作業的現場採樣監測，但優先級低於其他更高暴露風險的情境」。
