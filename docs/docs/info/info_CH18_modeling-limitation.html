核心公理

「所有模型都是錯的，但有些是有用的。」

- George E.P. Box

模型的價值在於其「足夠接近現實」以支持決策，而非追求完美。我們必須警惕將電腦輸出的「精密度」誤認為對現實的「準確度」。

實務陷阱：當模型假設遇上殘酷現實

模型的失敗往往不是程式錯誤，而是其核心假設在現實中不成立。這種「失敗」正是最有價值的診斷工具，能揭示潛在的工程或流程問題。

案例1：均勻混合室 (WMR) 模型

模型假設污染物完美混合，但現實中常因通風短路導致濃度差異巨大。

⚠️ 結論：模型預測值與實測值的巨大差異，直接指向了需要改善的通風工程問題，而非模型本身需要「校正」。

案例2：近場/遠場 (NF/FF) 模型

強力風扇打破了近/遠場的區隔，若模型忽略此效應，將嚴重高估近場濃度。

⚠️ 結論：更複雜的模型不一定更好。當現實情況違反其核心假設時，簡單模型可能更接近真相。

溝通的藝術：成為數據的轉譯者

原始數據並非資訊。職衛專家的核心價值在於將複雜的計算結果，轉譯為不同利害關係人能理解並據以行動的洞見。

利害關係人關注點分析

成功的溝通始於理解受眾。不同群體關心的重點截然不同，我們的報告也必須投其所好。

風險溝通的 paradigm shift：從「合格/不合格」到「機率性」理解

現代貝氏分析工具讓我們能更誠實、更直觀地溝通風險。我們不再問「是否超標？」，而是問「超標的機率有多大？」，這支持更精細的決策。
