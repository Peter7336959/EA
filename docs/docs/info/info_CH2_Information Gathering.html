① 關鍵資訊：化學物質特性

首要任務是了解化學品本身。除了分子量、密度等基本性質，最關鍵也最具挑戰性的參數是**逸散率 (G)** — 化學物質進入空氣中的速率。

範例：質量平衡法估算逸散率 (G)

透過物料耗用與廢棄量，可反推出平均逸散率。

② 關鍵資訊：空間環境特性

污染物所在的物理空間特性，直接影響其濃度分佈。必須精確掌握房間體積、通風狀況與空氣混合模式。

  • 📐

    房間體積 (V)

    需扣除設備佔據的「實佔體積」，以獲得更準確的空氣容積。

  • 💨

    通風 (Q)

    移除或稀釋污染物的關鍵。可透過現場量測、藍圖或文獻值取得。

  • 🔄

    空氣混合

    「完全混合」假設提供平均濃度，而「二區模式」更能反映靠近污染源的高濃度「近場」。

③ 關鍵資訊：勞工活動模式

最終評估的是「人」的暴露量。即使在完全相同的環境中(可能同一SEG)，不同的工作習慣也會導致暴露量出現顯著差異。經驗豐富的勞工 B 因為在污染源附近（近場）停留時間較短，其日時量平均暴露量僅為新手勞工 A 的一半左右。

結論

成功的暴露評估模式，建立在系統性且嚴謹的資訊收集之上。務必記錄所有假設與資訊來源，使評估報告更加專業、可信。

學習重點回顧

  • 逸散率 (G) 是最關鍵也最難估算的參數。
  • 通風 (Q) 與混合狀況 決定污染物在空間中的傳輸與最終濃度。
  • 時間 (t) 是連結「環境濃度」與「個人實際暴露量」的橋樑。
  • 暴露評估是一個**迭代過程**：從保守的粗估開始，若有必要，再投入資源蒐集更精確的數據來完善模式。
