模式推估-暴露未蓋化學品容器

意外暴露正己烷(n-hexane)的案例研究

事件概要

一名工作人員發現化學儲存櫃中一瓶廣口的正己烷容器未加蓋，濃烈的氣味瀰漫。他立即蓋上瓶蓋並撤離，隨後向安全團隊報告。本分析將透過模型評估此事件中的暴露風險。

核心問題 1

打開櫃門瞬間的暴露濃度有多高？

核心問題 2

污染物擴散後，整個房間的濃度是多少？

核心問題 3

房間需要多久才能安全進入？

階段一：櫃內的最壞情況

使用飽和蒸氣壓模型計算，我們估計當櫃門打開時，內部空氣中正己烷的濃度達到了驚人的水平。這是一個立即性的高度危險狀況。

163,000
ppm

(相當於 575,000 mg/m³)

⚠️ 此濃度遠超所有職業暴露限值，並高於爆炸下限！

階段二：房間的初始污染

當櫃門打開，高濃度氣體瞬間釋放到房間。我們使用無通風推估模式估算了房間的平均濃度，並將其與職業暴露限值 (OELs) 進行比較。

圖表顯示，即使是最低估計的房間濃度，也遠超過了建議的8小時時間加權平均濃度限值 (TLV-TWA)。

階段三：安全進入時間

在汙染源（未蓋的瓶子）被移除後，房間的通風系統開始清除污染物。完全混合模式預測了濃度隨時間下降的趨勢，讓我們可以確定何時能安全返回。

模型預測，大約需要50分鐘，房間濃度才能降至安全的TLV-TWA標準以下。

結論與最佳實務：從源頭預防

學習重點

  • 飽和蒸氣壓是評估密閉空間最壞情況的關鍵工具。
  • 多模型分析提供了從瞬間到長期的完整風險圖像。
  • 安全不僅是濃度問題，還需考慮爆炸範圍等指標。

最佳實務：逸散性排放控制

許多實驗室忽略了來自溶劑瓶的微量但持續的「逸散性排放」。

✅ 推薦方案：

持續通風或開放通風。
