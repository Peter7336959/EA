第四章：打磨作業EA - (單頁摘要)

1. 核心情境與評估任務

  • 背景問題：一間木工廠因產能不足，廠長提議臨時使用一張**無局部排氣裝置（LEV）**的工作台，每天進行2小時的打磨作業。
  • 職衛任務：預測此臨時作業是否會產生不可接受的木塵暴露濃度。

2. 關鍵參數與評估模型

為進行評估，我們採用完全混合室模式（Well-Mixed Room Model），並估算了以下關鍵參數：
* 暴露標準：選用較嚴格的 ACGIH TLV 作為主要判斷基準：1.0 mg/m³。
* 粉塵產生率 (G)：估算出粉塵產生率 G 約為 2,170 mg/min。粉塵粒徑極小，假設其完全懸浮於空氣中。
* 房間資訊：房間體積 (V) 為 3,120 m³，總排氣量 (Q) 為 240 m³/min。

3. 風險分析與關鍵發現

模型評估假設是污染物均勻混合 (但，操作人員呼吸帶的實際濃度必定更高)。
* 穩態濃度 (Css)：若此無通風作業持續進行，房間平均濃度將達到 9.0 mg/m³，遠超所有職業暴露標準。
* 2小時作業的日時量平均濃度 (TWA-8)：2.25 mg/m³，是 TLV 的兩倍多。
* 濃度累積速度：風險累積極為迅速。模型顯示，作業開始後僅需 1.5 分鐘，房間的平均濃度就會超過 ACGIH 的建議標準。

4. 結論與專業建議

  • 明確的風險判斷：
    廠長提議的臨時方案是不安全的，會產生不可接受的木塵暴露濃度。

  • 建議措施：

    1. 臨時方案：可考慮採用附有整合式集塵裝置的砂輪機作為臨時措施，但仍建議進行現場測定確認其有效性。
    2. 最終解決方案：最理想的工程控制方法是添購一台新的下吸式排氣桌，從根本上解決問題。
