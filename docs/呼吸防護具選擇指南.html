呼吸防護具選擇的智慧決策

基於風險評估的五步驟科學指南

提醒：呼吸防護具是控制危害的最後一道防線

1第一步：危害辨識與暴露評估

選擇正確的防護具，始於深入了解您所面對的敵人。此步驟是整個評估的基石，必須鉅細靡遺。

危害物種類 🧪

精確指認空氣中的有害物，如甲苯、二氧化矽粉塵等。查閱化學品安全資料表 (SDS) 是關鍵。

物理型態 💨

是固體/液體微粒（粉塵、燻煙、霧滴）還是氣體/蒸氣？這決定了過濾元件的選擇。

暴露濃度 📈

透過作業環境監測，取得勞工實際暴露濃度。這是計算防護等級的核心數據。

特殊狀況 ⚠️

確認是否為缺氧環境 (O₂ < 19.5%) 或立即致危濃度 (IDLH)，此時必須使用供氣式防護具。

2第二步：決定所需防護等級

了解危害強度後，需計算要將其降低多少倍才能達到安全標準，這就是「危害比」的概念。

計算公式：

危害比 (HR) = 作業環境中有害物濃度 / 該物質的容許暴露濃度 (PEL)

選擇原則：指定防護係數 (APF) ≥ 危害比 (HR)

案例 1：有機溶劑作業

甲苯濃度 150 ppm (PEL: 50 ppm)，危害比為 3。需選擇 APF ≥ 3 的防護具。

案例 2：人造石切割

二氧化矽濃度 1 mg/m³ (PEL: 0.05 mg/m³)，危害比為 20。需選擇 APF ≥ 20 的防護具。

根據計算出的 APF 值與危害物型態，從眾多防護具中，篩選出符合防護需求的類型。

4第四步：考量適用性因素

防護具光是「防護等級足夠」還不夠，還必須「適用」。一個勞工不願意或無法正確配戴的防護具，其防護效果等於零。

👤 使用者因素

  • 密合度：臉型、鬍鬚、疤痕都會影響密合。
  • 生理狀況：心肺功能、幽閉恐懼症等需評估。
  • 個人需求：是否需配戴眼鏡或頻繁溝通。

🛠️ 任務因素

  • 工作強度：高強度工作應考慮 PAPR 或供氣式。
  • 持續時間：長時間配戴需考量舒適性。
  • 視野需求：確保面鏡視野清晰且寬廣。

🌍 環境因素

  • 高溫高濕：影響舒適度與濾毒罐壽命。
  • 活動範圍：供氣式管線可能受限，PAPR 機動性高。
  • 與其他 PPE 衝突：需與安全帽、護目鏡等整合測試。
▼

5第五步：執行密合度測試 (Fit Testing)

這是確保緊密式防護具有效的最終驗證，也是法規的強制要求。每年、首次使用、更換型號或臉型變化時都必須執行。

定性密合度測試 (QLFT)

透過甜味/苦味測試劑，由使用者主觀判斷是否聞到味道。適用於 APF ≤ 10 的半面體防護具。

👍 / 👎

定量密合度測試 (QNFT)

利用儀器測量面體內外微粒濃度比值，得出客觀的密合係數。適用於所有緊密式防護具，為黃金標準。

使用者密合檢點 (User Seal Check)

這不是密合度測試！是使用者「每次」配戴時都必須做的簡易檢查（正壓與負壓檢點）。

重要注意事項

🎯 最後防線

永遠優先考慮工程控制。

🎓 教育訓練

成功的關鍵，確保使用者知能合一。

🔧 維護與儲存

定期檢查、清潔，並妥善存放。

📅 濾罐更換計畫

法定義務，絕不可憑感覺更換。
